# Rule-Based Chatbot 🤖

A simple chatbot built with Python using predefined rules.  
This project demonstrates the basics of natural language processing and conversation flow using `if-else` statements and pattern matching.

---

## 📂 Project Structure
chatbot-project/
├── chatbot.py          # Main chatbot script
├── rules.py            # Predefined rules and responses
├── utils.py            # Helper functions (optional)
├── requirements.txt    # Dependencies (can be empty for basic version)
└── README.md           # Documentation
