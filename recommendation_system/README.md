# 🎬 Recommendation System

A simple movie recommendation system built with Python using **content-based filtering** and **collaborative filtering** techniques.  
This project demonstrates how to suggest items (movies, books, or products) based on user preferences.

---

## 📂 Project Structure
recommendation-system/
├── recommender.py       # Main program logic
├── data.py              # Sample dataset (movies, genres, user ratings)
├── requirements.txt     # Dependencies (optional for advanced version)
└── README.md            # Documentation
